# Shift - Social Opinion Matching App

A Flutter mobile application that connects people through shared opinions. Users respond to statements and get matched with others who think similarly.

## Features

- **Opinion Voting**: Respond to statements with agree/disagree
- **Location-Based Matching**: Find people nearby with similar views using Geolocator
- **Spiral Animation**: Visual matching system showing proximity to potential matches
- **Real-time Database**: Firebase integration for instant updates
- **Conversation Guides**: Help users start meaningful conversations

## Tech Stack

- **Flutter** - Cross-platform mobile development
- **Dart** - Programming language
- **Firebase** - Backend and real-time database
- **Geolocator** - Location services
- **Figma** - UI/UX design
